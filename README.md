## Deploy your own
```
git clone https://github.com/Intai/bdux-examples.git
cd bdux-examples/infinite-scroll

npm install
gulp dev

open http://localhost:8080
```

```
git clone https://github.com/Intai/bdux-examples.git
cd bdux-examples/infinite-scroll

docker-compose up

open http://localhost:8080
```
